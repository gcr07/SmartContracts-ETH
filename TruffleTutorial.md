# Truffle

### Instalar Truffle

> https://www.npmjs.com/package/truffle/v/5.4.10-alpha.1

Instalar nodejs para poder usar truffle

```
node -v
npm -v
npm install -g truffle
```

##  Verificar Instalacion 

```
truffle version
```
